配置app快捷键

  1. 创建一个Bank Workflow，或下载一份别人的配置bestswifter/my-workflow

  2. 进入上述workflow，右键 -> Triggers -> Hotkey，键入待用快捷键(Object Key)

  3. 进入上述workflow，右键 -> Actions -> Launch Apps/Files，使用alfred搜索app，用鼠标左键按住app记录，将app拖入到启动软件框中，并勾选Toggle visibility for apps(Object App)

  4. 鼠标左键从Object Key图块的尖角出连线到Object App

(第3、4步可以颠倒，从Object Key图块的尖角拖出的连线如未连到某个Object App，将弹出创建窗口)
